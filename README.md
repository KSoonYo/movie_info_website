**BACK-END**

- Django DRF





## 팀원 정보 및 업무 분담 내역



김순요: **front-end**

- 웹 UI 컴포넌트 구성
- 웹 UX 흐름 구성
- django API 서버와 통신
- 기본 스타일링
- 추천 알고리즘 구현



이재만: **back-end**

- django API 서버 구현
  - TMDB API 데이터 요청(현재 상영 영화 / 인기 영화)
  - DB 구축
- Client 서버와 통신
- 회원가입 기능 구현
- 로그인 기능 구현(토큰 발급)
- 추천 알고리즘 구현
- 기타 로직 구현(좋아요, 추천 등)
